General

What does this do?

CinderSelf takes a sentence template and inserts your chosen name and pronouns into it, in order for you to see how they sound in a sentence.

Technical

What tools are used to make this?

Visual Studio Code, with the StandardJS plugin to ensure the JavaScript is clean and consistent. I use GitHub Desktop as my Git client.
